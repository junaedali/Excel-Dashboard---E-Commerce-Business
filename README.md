# E-Commerce Sales Performance Dashboard

## Project Overview

This project presents an interactive **E-Commerce Sales Performance Dashboard** created using Microsoft Excel.  
The dashboard analyzes sales, orders, customer segments, regions, order status, and sales channels.

## Dashboard Features

- Category-wise sales analysis
- Division-wise performance filtering
- Monthly sales and order trend analysis
- Gender-wise sales comparison
- Age group-based customer analysis
- Order status analysis
- Top 5 city performance analysis
- Channel-wise sales contribution analysis

---

## Business Insights

- Total sales reached **21,176K**.
- Total orders were **31,047**.
- Average sales per order was **682**.
- Women customers contributed more than men.
- Millennial customers showed the highest contribution.
- Chittagong was one of the top-performing cities.
- Pickaboo appeared as the highest-performing sales channel.
- March showed the strongest monthly performance.
- Sales declined toward the end of the year.
- Delivered orders dominated the order status section.
